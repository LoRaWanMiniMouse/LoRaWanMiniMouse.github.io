template<template< class R > class T, class RADIOTYPE>
class LoraWanObject< T, RADIOTYPE >

An absolutely minimalistic LoRaWAN Class A stack implementation .

Remarks
In future implementation the constructor will contain : APPSKey, NWSKey, DevAdrr mandatory for APB devices DevEui, AppEUI, APPKey mandatory for OTA devices OTA or APB Flag. In future implementation A Radio objet will be also a parameter of this class.

void LoraWanObject< T, RADIOTYPE >::SetDataRateStrategy ( eDataRateStrategy  adrModeSelect)

SetDataRateStrategy of the devices.

Remarks
Refered to the dedicated chapter in Wiki page for detailed explanation about implemented data rate choice (distribution data rate).
The current implementation support 4 different dataRate Strategy : STATIC_ADR_MODE for static Devices with ADR managed by the Network MOBILE_LONGRANGE_DR_DISTRIBUTION for Mobile Devices with strong Long range requirement MOBILE_LOWPER_DR_DISTRIBUTION for Mobile Devices with strong Low power requirement JOIN_DR_DISTRIBUTION Dedicated for Join requests
